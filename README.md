## Running Locally


```sh-session
git clone https://github.com/sayz-rio/sayz.us.git
cd sayz.us
```


Create a `.env` file similar to [`.env.example`](https://github.com/sayz-rio/sayz.us/blob/main/.env.example).

Then install dependencies and run the development server:
```sh-session
pnpm install
pnpm dev
```
